# أوراد أحلى شباب

منصة إلكترونية احترافية لإدارة ومتابعة المتطوعين والمشاريع الإسلامية.

## المميزات

- 🔐 **نظام تسجيل دخول آمن** مع Supabase Auth
- 👥 **إدارة المستخدمين** (متطوعين، مشرفين، مديرين)
- 👨‍👩‍👧‍👦 **فصل البيانات** حسب النوع (ذكور/إناث)
- 📝 **نظام فورمز متقدم** مع أنواع أسئلة مختلفة
- ⭐ **نظام تقييم ونقاط** لكل إجابة
- 📱 **تصميم PWA** يعمل كتطبيق موبايل
- 🎨 **تصميم عصري** متجاوب مع جميع الأجهزة
- 🗣️ **دعم كامل للغة العربية** (RTL)

## الأدوار

| الدور | الوصف | الصلاحيات |
|-------|-------|----------|
| **متطوع** | مستخدم عادي | يرى بياناته فقط ويملأ الفورمز |
| **مشرف** | مشرف على فئة | يرى بيانات نفس النوع فقط |
| **مدير** | مدير النظام | يرى كل البيانات والإحصائيات |

## أنواع الأسئلة المدعومة

1. **نص** - إجابة نصية قصيرة
2. **نص طويل** - إجابة نصية متعددة الأسطر
3. **اختيار واحد** - اختيار إجابة واحدة
4. **اختيار متعدد** - اختيار أكثر من إجابة
5. **تقييم** - تقييم من 1-5
6. **تصنيف** - ترتيب العناصر
7. **مصفوفة** - أسئلة متعددة

## التقنيات المستخدمة

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend**: Supabase (Auth, Database, Storage)
- **Deployment**: GitHub Pages
- **PWA**: Service Worker, Manifest

## التثبيت

```bash
# تثبيت الاعتماديات
bun install

# تشغيل في وضع التطوير
bun run dev

# البناء للإنتاج
bun run build
```

## إعداد Supabase

1. أنشئ مشروع جديد في [Supabase](https://supabase.com)
2. انسخ الـ URL و Publishable Key
3. أنشئ ملف `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=sb_xxx
```

4. نفذ SQL Schema في Supabase SQL Editor
